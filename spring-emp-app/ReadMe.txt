A simple Spring Context project using crud operation.

Create the Following entity class
Employee(id, name, phone, designation, email and password)


Perform the Following tasks
1.Save Employee

2.Update Employee

3.Find Employee BY ID

4.Verify Employee by email and password

5.Verify Employee by phone and password

6.Verify Employee by Id and Password

7.Find Employee by salary

8.Find Employee by Designation

9.Find Employee by Name

10.Find Employees between a salary range
